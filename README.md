## The Story

Most portfolios are just lists of jobs. This one is a design argument.

Every project here — whether it was built in a Maluku island school with no electricity, prototyped in a Monash studio, or coded independently at home — comes back to the same belief: **the best learning design works for the learner with the fewest resources first, then scales upward.**

This repository is the living home of that work. It contains my professional experience, academic background, selected projects from formal roles, and personal projects I built to push what I know further.
